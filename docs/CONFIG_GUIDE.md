# Гайд по настройке Биомов и Дистанции

Файл конфигурации: `config/shadowgrid_biomes.json` (Создается после первого запуска игры).

## Формат конфига

Теперь вы можете указывать не только вес (частоту), но и **минимальную дистанцию** от спавна (в секторах).

### Пример

```json
{
  "biome_weights": {
    "minecraft:plains": { 
        "weight": 10, 
        "min_distance": 0 
    },
    "minecraft:jungle": { 
        "weight": 8, 
        "min_distance": 5 
    },
    "minecraft:mushroom_fields": { 
        "weight": 2, 
        "min_distance": 10 
    }
  }
}
```

### Параметры

* **weight**: Вес биома. Чем больше число, тем чаще встречается.
* **min_distance**: Минимальное количество секторов от спавна (0,0), начиная с которого биом может появиться.
  * `0`: Может появиться на спавне.
  * `5`: Появится только после 5-го сектора (примерно 1280 блоков от центра).

### Как настроить

1. Запустите игру один раз, чтобы файл создался (если его нет).
2. Откройте файл любым текстовым редактором.
3. Отредактируйте значения.
4. Перезапустите игру (или используйте команду перезагрузки конфига, если добавлена, но надежнее перезапуск).

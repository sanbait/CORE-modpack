# Shadow Grid Mod

Этот мод добавляет в Minecraft механику секторов и прогрессии.

## Основные Фичи

- **Сетка Мира:** Весь Overworld разделен на сектора 512x512 блоков.
- **Генерация Биомов (FIXED):** Используется кастомный генератор для создания "квадратных" биомов.
- **Барьеры:** Границы секторов могут быть закрыты (черный туман, партиклы).
- **Разблокировка:** Сектора открываются за Lux Crystal.

## Установка

1. Соберите мод: `gradlew build`
2. Скопируйте jar из `build/libs` в папку `mods`.
